# Customer Analytics Project (PostgreSQL + Power BI)

## 📊 Descripción
Proyecto de análisis de clientes para un negocio retail. Se construyó una base de datos desde cero, se generaron datos con Python y se realizaron análisis en SQL.

## 🧠 Objetivo
- Identificar clientes más rentables
- Detectar clientes inactivos
- Analizar ventas por categoría
- Calcular ticket promedio
- Analizar tendencias mensuales

## 🛠️ Tecnologías
- PostgreSQL
- Python (Faker, psycopg2)
- Power BI

## 📂 Estructura
- database/schema.sql
- database/analysis_queries.sql
- generate_data.py
- powerbi/dashboard.pbix

## 📈 Insights
- Se identificaron los clientes con mayor contribución de ingresos
- Se detectaron clientes con riesgo de abandono (>90 días)
- Se analizaron categorías con mayor facturación
- Se observó la tendencia mensual de ventas
